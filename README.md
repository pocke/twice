twice
============


Installation
-----------

```sh
go get github.com/pocke/twice
```


Usage
-----------

```sh
$ echo -n 'foobar' | twice # => foobarfoobar
$ sl | twice # Run SL twice
$ cowsay meow | flowterm | twice
```

Recommend
----------

- [Flowterm](https://github.com/pocke/flowterm): String flows in a terminal.
